Installation d'un service microsoft RDS

Le but de ce TP était d'installer un service de bureau à distance sur windows sever

Il était demandé de mettre en place les service avec trois machines Windows server et une cliente sous Windows 10:
pour les windows server nous avons un serveur Active Directory, un serveur web et un broker

Le but étant de faire un groupe de serveur et au final d'ouvrir un traitement de texte et quelques applications classiques par le service de bureau à distance
